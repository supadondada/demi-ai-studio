STEP 1｜AI入門編

AIの始め方完全ガイド
今日から無料で使えるようになる

この記事では、AIに一度も触れたことがない方を対象に、「どのAIを選ぶか」→「登録する」→「最初の質問をしてみる」までを順番に解説します。読み終わる頃には、あなたも「AIを使える人」です。

1. AIでできること — まず全体像をつかもう

ここで言う「AI」とは、主に生成AI（ジェネレーティブAI）のこと。日本語で話しかけるだけで、文章・画像・動画・プログラムなどを作ってくれる道具です。たとえば、こんなことが今日からできます。

  • 文章を書いてもらう — メール、企画書、ブログ記事、お詫び文、スピーチ原稿
  • わからないことを教えてもらう — 専門用語の解説、勉強のサポート、家庭教師代わり
  • アイデアを出してもらう — 商品名の案100個、旅行プラン、献立の提案
  • 作業を代わりにやってもらう — 長文の要約、英語の翻訳、表の整理、文章の校正
  • 画像や動画を作ってもらう — イラスト、ロゴ案、SNS用画像、ショート動画（→ 応用編で解説）
ポイント：AIは「検索エンジンの上位互換」ではなく「優秀なアシスタントを1人雇う」イメージです。調べるだけでなく「作らせる」「手伝わせる」ことができるのが最大の違いです。

2. 三大対話AIの選び方 — ChatGPT・Claude・Gemini比較

2026年現在、個人がまず使うべき対話AIはこの3つです。すべて無料で始められるので、深く悩む必要はありません。それぞれの得意分野を知っておきましょう。

三大対話AIの比較（無料プランあり・スマホアプリあり）
サービス名提供元得意なことこんな人におすすめ
ChatGPT OpenAI 利用者が最も多く情報も豊富。会話・画像生成・音声対話までオールラウンド 迷ったらまずコレ。世間で話題の「AI」を体験したい人
Claude Anthropic 長文の読み書き・自然で丁寧な日本語・資料の分析・プログラミング支援 文章の質を重視する人、仕事の文書やnote執筆に使いたい人
Gemini Google Google検索・Gmail・Googleドキュメントとの連携、最新情報の検索 Googleのサービスを日常的に使っている人

結論：まずChatGPTかClaudeのどちらかで始めて、慣れたら3つとも無料で併用するのがおすすめです。同じ質問を投げ比べると、それぞれの個性がよくわかります。

3. アカウント登録の手順 — 5分で完了

ここではChatGPTを例に説明しますが、ClaudeもGeminiもほぼ同じ流れです。必要なものはメールアドレスだけ（GoogleアカウントやApple IDでも登録できます）。

  1. 公式サイトまたは公式アプリを開く

    パソコンなら「ChatGPT」で検索して chatgpt.com へ。スマホならApp Store / Google Playで「ChatGPT」を検索し、提供元が「OpenAI」の公式アプリをインストールします（偽アプリに注意）。Claudeは claude.ai、Geminiは gemini.google.com です。

  2. 「サインアップ（新規登録）」を選ぶ

    「Googleで続ける」を選ぶと最短です。メールアドレスで登録する場合は、届いた確認メールのリンクをクリックします。

  3. 名前と生年月日を入力する

    ニックネームでも構いません。生年月日は年齢確認のために必要です。

  4. 最初のメッセージを送ってみる

    画面下の入力欄に「こんにちは。AIを初めて使います。何ができるか教えてください」と打って送信。返事が来たら登録完了です！

注意：登録時にパスワードを設定した場合は必ず控えておきましょう。また、SNS広告などから飛べる「非公式の類似サービス」には課金しないよう、URLが公式（chatgpt.com / claude.ai / gemini.google.com）であることを確認してください。

4. 最初に試したい3つの使い方（コピペOK例文つき）

登録できたら、まずこの3つを試してください。AIの実力が5分で体感できます。

① 文章を書いてもらう

コピペして使える例文 取引先に送る、打ち合わせ日程変更のお願いメールを書いてください。 ・相手：株式会社〇〇の田中様 ・元の日程：7月15日（水）14時 ・希望：7月17日（金）の午後 ・トーン：丁寧だが堅すぎない

箇条書きで条件を伝えるだけで、そのまま送れるレベルのメールが数秒で完成します。

② 要約してもらう

コピペして使える例文 次の文章を、小学生にもわかる言葉で3行に要約してください。 （ここにニュース記事や長いメールを貼り付け）

長い資料やニュースを貼り付けるだけ。「小学生にもわかるように」「箇条書きで」など、まとめ方も指定できます。

③ 相談相手になってもらう

コピペして使える例文 週末に家族（大人2人・小学生1人）で楽しめる東京近郊のお出かけ先を、 予算1万円以内で5つ提案してください。それぞれの魅力も一言添えてください。

旅行プラン、献立、プレゼント選び、キャリア相談まで。「追加で質問を重ねられる」のがAIの強みです。「2番の案をもっと詳しく」のように会話を続けてみましょう。

5. 無料プランと有料プランの違い

結論から言うと、入門のうちは無料プランで十分です。有料プラン（各サービスとも月額20ドル前後＝約3,000円）を検討するのは、次のような状態になってからで構いません。

無料プラン有料プラン（月額約3,000円）
基本的な会話◎ 十分使える◎
利用回数△ 混雑時や高性能モードに制限あり◎ 制限が大幅に緩和
AIの性能○ 標準モデル中心◎ 最高性能モデルを優先利用
画像生成・ファイル分析△ 回数制限あり◎ たっぷり使える

有料化の目安：「回数制限に毎日ひっかかる」「仕事や副業で使い始めた」——そうなったら月3,000円は簡単に元が取れます。詳しい活用法は実業務編と副業編で解説しています。

6. 安全に使うための注意点

  • 個人情報・機密情報を入力しない — 氏名・住所・パスワード・会社の内部資料などは入力しない。入力内容がAIの学習に使われる設定になっている場合があります（設定でオフにできます）。
  • AIの回答を鵜呑みにしない — AIは時々、もっともらしい間違い（ハルシネーション）を言います。日付・数字・固有名詞・法律や医療の情報は必ず出典を確認しましょう。
  • 会社で使う前にルールを確認 — 勤務先にAI利用のガイドラインがある場合はそちらが優先です（→ 実業務編で詳説）。
  • 著作権に配慮する — AIが作った文章や画像を商用利用する際のルールはサービスごとに異なります（→ 副業編で詳説）。

よくある質問

AIを使うのにお金はかかりますか？

ChatGPT・Claude・Geminiはいずれも無料プランがあり、メールアドレスだけで今日から使えます。まずは無料で始めて、物足りなくなったら月額約3,000円の有料プランを検討すれば十分です。

パソコンがなくてもAIは使えますか？

使えます。3サービスともiPhone/Android向けの公式アプリがあり、スマホだけで登録から利用まで完結します。音声で話しかけられるのはスマホアプリならではの利点です。

AIに入力した内容は他人に見られませんか？

会話が他のユーザーに公開されることはありませんが、入力内容がAIの学習に使われる場合があります。個人情報や機密情報は入力しない、設定画面で学習利用をオフにする、といった対策をおすすめします。

ChatGPT・Claude・Geminiのどれから始めるべきですか？

迷ったら利用者が最も多いChatGPTがおすすめです。長い文章や丁寧な日本語が必要ならClaude、GoogleのサービスとつなげたいならGemini。すべて無料なので、同じ質問を投げ比べて自分に合うものを選ぶのが最善です。
