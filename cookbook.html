Claude Cookbookとは？
公式レシピ集の中身と使い方

AIの情報を追っていると出てくる「Cookbook（クックブック）」。名前のとおり料理のレシピ集のようなものですが、これは開発者向けの資料です。このページでは、それが何者で、誰に必要で、何が要るのかを包み隠さず解説します。「自分にはまだ早い」と判断できることも、このページの目的です。

① Claude Cookbookとは？ — 公式の「実例レシピ集」

Claude Cookbookとは、Claudeの開発元であるAnthropicが公式に公開している、実践的な使用例集です。GitHubというプログラマー向けのサイトで誰でも無料で閲覧できます。公式の説明文は次のとおりです。

「Claudeを使う、楽しく効果的な方法を紹介するノートブック／レシピ集」
— anthropics/anthropic-cookbook 公式リポジトリの説明より

例え話：料理本と同じ構造

料理のレシピ本に「肉じゃがの作り方」「カレーの作り方」が1品ずつ載っているように、Cookbookには「PDFを要約させる方法」「問い合わせを自動分類する方法」「画像を読ませる方法」が1本ずつ、動くコードとして収められています。

ゼロから考えるのではなく「似たレシピを探して、自分用に材料を差し替える」——これがCookbookの使い方です。

先に言っておきます：Cookbookの中身は9割以上がJupyter Notebook形式のPythonプログラムです。ChatGPTやClaudeのチャット画面のような、開いてすぐ使えるツールではありません。プログラミングをしない方が今すぐ開く必要はまったくありません。次のセクションで、あなたに必要かどうかを判定します。

② 誰向け？ — あなたに必要か判定する

このサイトは初心者向けですが、Cookbookは正直に言って初心者向けではありません。無理に勧めることはしません。次の表で自分の立ち位置を確認してください。

あなたの状況Cookbookは必要？代わりに読むべきページ
AIを使い始めたばかり 不要 AI入門編 — まずチャット画面に慣れる
プロンプトを上手くしたい 不要 応用編のプロンプト講座で十分
仕事でAIを時短に使いたい 不要 実業務編 — 既製ツールで足ります
AIを組み込んだアプリを作りたい ◎ 役立つ このページを読み進めてください
開発の副業を狙っている ○ 武器になる 副業編の開発代行と併せて
大事な考え方：Cookbookが扱うのは「プログラムからAIを呼び出す」世界の話です。これは基礎知識で解説したAPI——アプリ同士をつなぐ窓口——を使う領域。チャット画面でAIを使うのとは別のスキルであり、片方だけでもまったく問題ありません。

③ 必要なもの3つ — 前提条件を正直に

Cookbookのレシピを実際に動かすには、公式が挙げている次の3つが必要です。「見るだけ」ならブラウザだけでできます。

1. APIキー（無料登録・ただし従量課金）

APIキーとは、プログラムからAIを呼び出すためのパスワードのような文字列です。Anthropicのサイトで無料登録して取得できます。

料金の落とし穴：APIキーの取得自体は無料ですが、プログラムからAIを動かすと、使った分だけ課金される従量課金制です。チャット画面の月額プラン（約3,000円）とはまったく別の料金体系で、月額を払っていてもAPI利用分は別請求になります。少額から試せますが、この違いは必ず理解してから始めてください。

2. Python（プログラミング言語）

CookbookのコードはほぼすべてPythonで書かれています。Python自体は無料で、AIやデータ分析で最も使われている言語です。文法をひととおり知っている必要があります。

3. Jupyter Notebook（ノートブック形式の実行環境）

聞き慣れない名前ですが、「説明文とプログラムを交互に書けて、1ブロックずつ実行して結果を確認できる」ノートのことです。Cookbookのファイルの94%がこの形式（拡張子 `.ipynb`）です。

朗報：Jupyterは自分のPCに入れなくても、Google Colab（コラボ）という無料のWebサービスを使えばブラウザだけで動かせます。GitHubのノートブックをColabで開けば、インストール作業をほぼ飛ばせます。

④ 正しい始め方 — Cookbookは「最初に開く場所」ではない

ここが最重要です。Cookbook自身が、いきなり開かず先に別の資料を見るよう案内しています。実際、公式の「Prerequisites（前提条件）」欄には 「API Fundamentalsコースを見てほしい」と書かれています。

Anthropicは学習用のコース集を別に公開しており（`anthropics/courses`）、推奨順まで示されています。この順番を無視して先にCookbookを開くのが、挫折の典型パターンです。

  1. API Fundamentals（API入門）

    APIキーの取得、モデルの設定値、画像を含むプロンプト、応答の逐次表示など基礎の基礎。ここが出発点です。

  2. Prompt Engineering Interactive Tutorial（プロンプト対話型講座）

    プロンプト技術を手を動かしながら学ぶ講座。応用編で解説した「型」の、より体系的な版です。

  3. Real World Prompting（実践プロンプト）

    学んだ技術を、現実の複雑なプロンプトに組み込む方法。

  4. Prompt Evaluations（プロンプト評価）

    作ったプロンプトの品質を数値で測る方法。趣味と実務を分ける重要な工程です。

  5. Tool Use（ツール活用）

    AIに外部のツールを使わせる方法。基礎知識で触れたAIエージェントの土台になる技術です。

  6. → ここでようやくCookbook

    基礎ができた状態で、「やりたいこと」から逆引きする辞典として使う。これが本来の位置づけです。

料理で例えるなら：包丁の握り方も知らない人がレシピ本を開いても作れません。コース集が「料理教室」、Cookbookが「レシピ本」。教室→レシピ本の順が正解で、逆にすると挫折します。

⑤ 中身のカタログ — どんなレシピがあるのか

「自分の作りたいものがあるか」を判断できるよう、実際の分類を紹介します。

基本の能力（Capabilities）

  • 分類（Classification） — 問い合わせを種類ごとに自動仕分けする
  • RAG（検索拡張生成） — 自社の資料を読ませて、その内容に基づいて答えさせる
  • 要約（Summarization） — 長文を要点にまとめる

ツール活用・外部連携

  • ツール活用 — カスタマーサポート担当AI、電卓との連携、SQL（データベース）への問い合わせ
  • 外部サービス連携 — ベクトルデータベース（Pinecone）、Wikipedia、Webページの読み込み、Voyage AIによる埋め込み処理

マルチモーダル（画像を扱う）

  • Claudeの画像認識 — 画像の分析、グラフ・図表の読み取り、フォームからの情報抽出
  • 画像生成 — Stable Diffusionとの連携（→ 応用編の画像生成）

応用テクニック

  • サブエージェント — AIに補佐役のAIを使わせる
  • PDFのアップロードと要約 — 資料をそのまま読ませる
  • 自動評価 — AIの出力品質を自動で採点する
  • JSONモード — 出力をプログラムで扱える形式に固定する
  • コンテンツモデレーション — 不適切な内容を検知するフィルタ
  • プロンプトキャッシュ — 同じ前提文の再利用でコストを削減する
眺めるだけでも価値があります：コードが読めなくても、この一覧を見るだけで「AIってこんなことに使えるのか」という発想の引き出しが増えます。RAGやサブエージェントといった言葉に触れておくだけでも、AIのニュースの解像度が上がります。

よくある質問

Claude Cookbookとは何ですか？

Anthropicが公式に公開している、Claudeの実践的な使い方をまとめたサンプル集です。GitHubで無料公開されており、「分類する」「PDFを要約する」といったテーマごとに動かせるコードが1本ずつ入っています。ただし中身のほとんどはJupyter Notebook形式のプログラムで、開発者向けの資料です。

Cookbookを使うのにプログラミングは必要ですか？

必要です。ファイルの9割以上がPythonのJupyter Notebookで、動かすにはPythonの実行環境とAPIキーが要ります。プログラミングをしない方は無理に開く必要はありません。まずはチャット画面のAIを使いこなす方が先です。

APIキーとは何ですか？無料ですか？

プログラムからAIを呼び出すためのパスワードのような文字列です。取得は無料ですが、実際にプログラムからAIを動かすと使った分だけ料金がかかる従量課金制です。チャット画面の月額プランとは別の料金体系なので注意してください。

Cookbookはどこから始めればいいですか？

Cookbook自身が「先にAPI Fundamentalsコースを見てほしい」と案内しています。Anthropicは学習用のコース集を別に公開しており、API入門→プロンプト講座→実践→評価→ツール活用という推奨順があります。Cookbookはその後に開く「逆引き辞典」と考えるのが正解です。
